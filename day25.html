第25回：単方向リストの要素削除

00:00
単方向リストから、指定された値を持つインスタンスを「削除」したい。以下のプログラムを実行したとき、【 空欄 】に入る適切な代入文を答えよ。
💡 解答スピードアップのコツ（情報の絞り込み）
「削除したい値（target）は 30 である」と最初から分かっています。
そのため、whileループがどこで止まるかを真面目に一行ずつ追わなくても、「ループが終わったとき、削除対象の [30] を curr が指していて、その1歩手前の [20] を prev が指しているはずだ」 と一瞬で見抜くことができます！必要な部分だけに注目してみましょう。

【初期状態と状況】
・現在、リストには [ダミー] → [10] → [20] → [30] → [40] → NULL の順でインスタンスが並んでいます。
・削除したい値 target は 30 です。
・whileループによる探索が終了した直後、各ポインタは以下の位置で止まっています。

prev は [20] のインスタンスを指している
curr は 削除対象である [30] のインスタンスを指している

/* 初期設定 */ prev ← listHead（ダミーノード） curr ← listHead.next（10のインスタンス） target ← 30（削除したい値） /* 削除したいインスタンス（30）の手前までポインタを進める */ while (curr が NULL ではない) 且つ (curr.value ≠ target) prev ← curr curr ← curr.next endwhile /* 【注目】ここで [30] のインスタンスをリストから切り離す（削除する） */ 【 空欄 】
