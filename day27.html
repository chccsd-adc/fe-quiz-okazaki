双方向リストの途中（要素と要素の間）に、新しいインスタンスを挿入したい。以下のプログラムを実行したとき、【 空欄 】に入る適切な代入文を答えよ。
💡 2分で解くための絞り込みヒント
4本ある矢印のうち、すでに3本はプログラムによって繋ぎ終わっています。
残るは最後の1本、「右隣のインスタンス（30）から、新参者（25）へ戻る左向きの矢印（prev）」を繋ぐ処理だけです！右隣の要素（30）をどう表現するかに注目しましょう。

【初期状態と状況】
・現在、リストには [10] ⇄ [20] ⇄ [30] ⇄ NULL の順でインスタンスが並んでいます。
・ポインタ curr は [20] のインスタンスを指しています。
・ここに、新しく作ったインスタンス p（値は 25）を、[20] と [30] の間に挟み込みたいです。

/* 1. 新参者pから、前後（20と30）へ往復の矢印を伸ばす */ p.prev ← curr （pの左は20） p.next ← curr.next （pの右は30） /* 2. 左隣（20）から、新参者pへ右向きの矢印を伸ばす */ curr.next ← p （20の右はp） /* 3. 【注目】右隣（30）から、新参者pへ左向きの矢印（prev）を戻す */ 【 空欄 】 ← p
